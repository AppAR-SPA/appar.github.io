Url externo GET

/external/:provider_id

Obtener enlace de un producto externo, dando su id externa, es buscado en AppAR si tiene algún producto o colección vinculados y retornará el correspondiente.

Una id externa no puede estar asignada a un producto y una colección al mismo tiempo. La respuesta de esta petición debe ser agregada a la ruta del visor 3D: https://webviewer.appar.io o https://dev.webviewer.appar.io para entorno sandbox. Esta ruta es pública, no necesita Bearer Token en la cabecera.

Request Body

No tiene

Retorna

"/collection/1"

o

"/1"

Dependiendo de si es una colección o un producto

Errores

500 error_retrieving_product_provider Si hay un error buscando el proveedor externo
500 error_retrieving_collection_provider Si hay un error buscando el proveedor externo
500 no_provider_linked Si no hay ni un producto ni colección asociado a este id externo
500 error_retrieving_model Si hay un error buscando el modelo asociado al producto
500 model_not_found Si el producto asociado no tiene modelo 3D
