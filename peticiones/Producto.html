Producto de externo GET

/product?provider_product_id=prod5220044

Obtener el model_id de un producto externo de integración entregando la id de producto de la plataforma externa a la que está vinculado el producto AppAR

Esta ruta es pública, no necesita Bearer Token en la cabecera.

Request Body

No tiene

Retorna

{
    "model_id": 1
}

Errores

500 error_retrieving_product Si hay un error obteniendo los datos
500 product_not_found Si no se encuentra esa id externa vinculada a un producto AppAR
