Modelo 3D GET

/model/:model_id

Obtener datos de modelo 3D de un producto AppAR

Se debe entregar el model_id y no el product_id de un producto. Esta ruta es pública, no necesita Bearer Token en la cabecera.

Request Body

No tiene

Retorna

{
    "model_id": 1,
    "variant_id": 1,
    "android_model_url": "https://appar.com/product1.glb",
    "ios_model_url": "https://appar.com/product1.usdz",
    "status_id": 5,
    "ar_placement": "floor",
    "ar_scale": null,
    "ar_fixed_annotation": null,
    "poster_url": null,
    "created_at": "2021-11-11T15:08:47.000Z",
    "updated_at": "2021-11-11T15:08:47.000Z",
    "product_name": "Producto 1",
    "product_id": 3,
    "product_url": "https://tienda.com/producto-1",
    "store": "Tienda",
    "call_to_action": "Ver producto en tienda",
    "logo_url": null,
    "show_dimensions": false,
    "annotations": []
}

Errores

500 model_not_found Si el model_id entregado no existe
500 product_not_found Si no se encuentra un producto asociado al modelo 3d
500 product_not_available Si el producto y modelo existen pero no es accesible por motivos de pago

Agregar aumentación PUT

/:product_id/augmentation

Sumar una aumentación con timestamp al producto dado su id

No confundir product_id con model_id. Esta ruta es pública, no necesita Bearer Token en la cabecera.

Request Body

No tiene

Retorna

Nada

Errores

500 error_retrieving_product Si hay un error buscando el producto
500 product_not_found Si no se encuentra el producto con la id entregada
500 error_adding_augmentation Si hay un error al agregar la aumentación
